# Lesson 12: Kotlin Consumer App

## Description
Using published artifacts in a Kotlin application.

## Key Concepts
- Kotlin JVM plugin
- Application plugin with Kotlin
- Consuming Java libraries in Kotlin

## Runnable Examples
- Run `./gradlew run` (executes Main.kt, which uses the Library from lesson10).

## Prerequisites
- Run `./gradlew :lesson10-publishing:publish` first to publish the library.
