# Lesson 2: Setting Up a Gradle Project

## Description
Creating and initializing a new Gradle project.

## Key Concepts
- Project structure
- settings.gradle
- Gradle wrapper

## Runnable Examples
- Groovy DSL: Run `./gradlew build` (compiles the Java code).
- Kotlin DSL: Same command.

## Exercises
1. Add a new source file (e.g., `Greeting.java`) and rebuild the project.
   - Create the file in `src/main/java/`.
   - Run `./gradlew build` to verify.
