# Lesson 10: Publishing Artifacts

## Description
Sharing builds via repositories.

## Key Concepts
- Maven/Gradle publishing
- Repositories

## Runnable Examples
- Run `./gradlew -p lesson10-publishing publishToMavenLocal`.
- Run `./gradlew -p lesson10-publishing publish` (publishes to the configured remote repo at `../remote-repo`).

## Exercises
1. Configure publishing to a remote repo. (Completed: Added a local file-based repository at `../remote-repo` for demonstration.)
