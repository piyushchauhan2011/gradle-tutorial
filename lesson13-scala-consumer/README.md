# Lesson 13: Scala Consumer App

## Description
Using published artifacts in a Scala application.

## Key Concepts
- Scala plugin
- Application plugin with Scala
- Consuming Java libraries in Scala

## Runnable Examples
- Run `./gradlew -p lesson13-scala-consumer run` (executes Main.scala, which uses the Library from lesson10).

## Prerequisites
- Run `./gradlew -p lesson10-publishing publish` first to publish the library.
